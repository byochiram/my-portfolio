# Rosidah Rahmati Portfolio

Personal portfolio website showcasing profile, selected projects, skills, certifications, and contact information.

## Live Website

https://my.rosispace.workers.dev/

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- TanStack Start
- Cloudflare Workers

## Sections

- About
- Projects
- Skills
- Certifications
- Contact

## Getting Started

Clone this repository:

```bash
git clone https://github.com/byochiram/my-portfolio.git
```

Go to the project directory:

```bash
cd my-portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Deploy to Cloudflare Workers:

```bash
npx wrangler deploy
```
